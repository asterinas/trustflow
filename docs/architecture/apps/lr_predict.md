# LR预测

使用给定的线性/逻辑回归模型对数据进行预测。

## 组件定义

1. 参数
    (1) pred_name: 预测值的列名。
    (2) save_label: 输出结果是否包含标签列，true表示保存。
    (3) label_name: 标签列的名称，默认为“label”。
    (4) save_id: 输出结果是否保存ID列，true表示保存。
    (5) id_name： ID列的名称。
    (6) col_names: 可选，输出指定的列到结果中，默认为空。
2. 输入：待预测的数据以及LR模型。
3. 输出：预测结果。
